The Rise of Agentic AI

What are AI Agents?
AI agents are programs or platforms that possess high autonomy and agency, enabling them to make decisions and act to achieve specific goals, often with reduced human intervention. Unlike traditional AI, which typically functions as a tool for narrow tasks, agentic AI systems evolve into team members by managing and executing end-to-end processes. These agents utilize advanced reasoning models to break down complex goals, find solutions, and continuously learn and adapt.

Adoption is rising fast!
Agentic AI represents a rapidly emerging technological trend expected to reshape enterprise efficiency and growth. This technology is projected to generate up to $450 billion in economic value across surveyed countries by 2028 through revenue uplift and cost savings. Adoption momentum is clearly strong: 93% of leaders believe that successfully scaling AI agents in the next 12 months will provide a competitive advantage. Currently, 14% of organizations have implemented AI agents at partial or full scale, with another 23% running pilot projects.

Some Challenges
However, organizations face significant challenges that limit the deployment of highly autonomous systems. Trust in fully autonomous AI agents is declining, with only 27% of organizations expressing confidence, a drop from 43% twelve months ago. Concerns are widespread regarding ethical and safety risks, such as data privacy and algorithmic bias. Furthermore, over 80% of organizations lack mature AI infrastructure, and fewer than one in five report high levels of data readiness, which significantly impedes the ability to scale agentic systems effectively.

Technical Description

AWS Infrastructure

The application and the database are hosted on AWS using an EC2 Instance in US East Region.

  • Application Instance Type: t3.small
  • Operating System: Amazon Linux 2023
  • Web Server: Nginx
  • Back-End Port Number: 3000
  • Front-End Port Number: 80
  • Database Used: MySQL, hosted in Amazon Aurora RDS

    • Application

    Back-End
    API is developed using Python and hosted on AWS EC2 instance.

  • Python Package for REST API: FastAPI
  • Hosted using nohup and uvicorn packages

    • Front-End
    The front-end is developed using Angular 21 and served using nginx on AWS EC2 instance.

    Features Implemented

    The following features are implemented in the application:

  • Single Page Application using Angular UI, Python API and MySQL Database
  • Login Page
  • Dashboard (homepage) with Insights on Rise of AI Agents and Technical Details
  • Summary Page with 2 charts
  • Report Page with a multi-chart view
  • Login and Logout Functionalities using JWT Tokens
  • Code check-in excludes build-related files using gitignore
    • Additional work:
  • 3-tier Service Oriented Architecture design is implemented
  • AuthGuard, LoginGuard, a common AuthService, etc. are implemented to comply with proper Angular design.
  • Favicon is implemented
  • User's password is hashed while signing up (only API endpoint, no UI). Hashed password is verified during Login process.
